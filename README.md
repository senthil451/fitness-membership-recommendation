# Fitness Membership Recommendation System

## Project Overview
This project predicts the membership price and recommends a suitable fitness plan based on user inputs.

## Features
- Predicts membership price using Linear Regression
- Recommends a plan using Random Forest
- Categorizes users into Basic, Medium, or Premium plans

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Input Features
- Age
- Gender
- Goal_Type
- Weekly_Visits
- Diet_Plan
- Personal_Training
- Previous_Surgery

## Output
- Predicted Price
- Recommended Plan
- Category

## How to Run
1. Install required libraries
2. Run the Python script
3. Enter user details
4. View predictions
